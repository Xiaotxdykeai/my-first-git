#  计算机基础

- **数据结构**: 研究数据是怎么存储的，怎么着怎么处理
- **操作系统**: 研究计算机到底怎么管理各种程序运行的
- **计算机网络**: 互联网
- **计算机组成原理**: 电脑里面的硬件到底是做什么工作的，到底是有什么作用的

## 书籍推荐

- **数据结构**: 《大话数据结构》《数据结构与算法分析》
- **操作系统**: 《操作系统导论》《深入理解计算机系统》
- **计算机网络**: 《计算机网络：自顶向下方法》
- **计算机组成原理**: 《计算机组成与设计》《深入理解计算机系统》

>数据结构-操作系统-计算机网络-计算机组成原理，是我学习的顺序
